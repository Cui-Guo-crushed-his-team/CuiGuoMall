# RPC Module
```
├── idl // Interface Description Language (IDL) definition
│   ├── proto // protobuf protocol interface definition
│   │   ├── auth.proto
│   │   ├── cart.proto  
│   │   ├── checkout.proto
│   │   ├── order.proto
│   │   ├── payment.proto
│   │   ├── product.proto
│   │   ├── user.proto
│   └── ... // If other IDL files are needed, they will be placed at the same level
├── kitex_gen // kitex automatically generated code,should not be modified
│     ├── auth
│     ├── cart
│     └── ...
├── rpc // The rpc client code generated by cwgo,should be further refined after it is generated
│     ├── auth
│     ├── cart
│     └── ...
```
